Chess Complexity Calculator

Enter a FEN string to calculate the complexity of the position.

What can you expect from this site in the future?

My goal is to create a suite of tools which allow you to evaluate the complexity of a position, evaluate a players prformance in a game relative to the complexity of the postitions they encountered, evaluate a performance across an entire performance, and use those tools to help find outliers (e.g., cheating).

You can see the code for all this work here: Elocator GitHub. Please consider contributing!
